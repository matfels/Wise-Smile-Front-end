# 🦷 WiseSmile Front-end

O **WiseSmile Front-end** é a interface web do sistema de gestão odontológica, construída para proporcionar uma experiência fluida no gerenciamento de consultas, pacientes e especialistas.

## 🚀 Tecnologias Utilizadas
- **Framework:** Angular 17+
- **Estilização:** SCSS
- **Gerenciamento de Estado:** Services Angular
- **Comunicação:** HttpClient (Consumo da API Rest WiseSmile)

## ⚙️ Funcionalidades
- **Autenticação:** Interface de login integrada com JWT.
- **Gestão Clínica:** Telas para CRUD de pacientes, dentistas e especialidades.
- **Agendamento:** Interface para novo agendamento e visualização de consultas.
- **Responsividade:** Design adaptado para dispositivos desktop e mobile.

## 📋 Pré-requisitos
Para rodar este projeto, você precisará de:
- [Node.js (LTS)](https://nodejs.org/)
- [Angular CLI](https://angular.io/cli)
